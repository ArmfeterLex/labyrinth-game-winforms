# Labyrinth Game WinForms

Простая игра-лабиринт на C# Windows Forms.

<img width="771" height="621" alt="Снимок экрана 2026-03-01 190057" src="https://github.com/user-attachments/assets/64745b17-edaf-41ac-83cf-d6560525559c" />

## Функциональность

- Лабиринт из label-стенок
- При касании стенки — возврат на старт
- При касании финиша — поздравление
- Стартовая позиция курсора при загрузке

## Интерфейс

- label29 — стена (возврат на старт)
- label36 — финиш (поздравление)
- Остальные label — стены лабиринта

## Куда можно встроить

- В игровые мини-проекты
- В обучение работе с событиями мыши
- В демонстрацию обработки MouseEnter

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Координаты возврата подобраны под стартовую позицию курсора.
